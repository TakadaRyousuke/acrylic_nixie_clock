# arrange LED in 8*8 matrix
# 2020/01/01 powertimer811
#　このスクリプトは指定した電子部品をマトリクス状に並べる為の物です。

#--- 使い方 ---#
# 以下の引数を指定して、関数を使ってください。

#--- 引数説明 ---#
# matrix_size     :: マトリクス状に並べる際の、マトリクスサイズです。[x, y](個)に並べます。
# grid_moment     :: 電子部品の間隔です。[x, y](mm)の間隔で並べます。
# offset          :: 1個目の電子部品の座標です。[ x, y](mm)
# parts_head      :: 電子部品のアルファベットを指定してください。例えば、D1~D8を並べたければ、この引数には、"D"と指定してください。
# parts_num_range :: 電子部品の数字を指定してください。例えば、D1~D8を並べたければ、この引数には、[1, 8]と指定してください。
# angle           :: 電子部品の回転角を指定してください。(度)
